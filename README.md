# maximilianfellner.eu 2

Personal website of Maximilian Fellner.

## Development

Install dependencies:

```
npm install
```

Build static files:

```
gulp build
```

Start with live reloading:

```
nodemon --harmony
```

The web service should now be running on [http://localhost:3000]().
